# Sound Files

This directory should contain the following sound files:

## Default Theme
- `default-start.mp3` - Sound played when starting a session/break
- `default-end1.mp3` - First sound played when completing a session/break
- `default-end2.mp3` - Second sound played after completion (with 400ms delay)

## Soft Theme
- `soft-start.mp3`
- `soft-end1.mp3`
- `soft-end2.mp3`

## Loud Theme
- `loud-start.mp3`
- `loud-end1.mp3`
- `loud-end2.mp3`

## Recommendations

For production use, you can:

1. **Create your own sounds** using audio editing software
2. **Use royalty-free sounds** from sites like:
   - Freesound.org
   - Zapsplat.com
   - Mixkit.co

3. **Suggested sound types**:
   - Start: Short, gentle chime or bell (0.5-1 second)
   - End1: Completion bell or notification (1-2 seconds)
   - End2: Softer follow-up chime (0.5-1 second)

## Format Requirements
- Format: MP3
- Bitrate: 128-192 kbps recommended
- Sample Rate: 44.1 kHz
- Duration: 0.5-2 seconds for optimal UX
